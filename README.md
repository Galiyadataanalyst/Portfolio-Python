# Анализ активности игровых гильдий

## Описание проекта

В проекте проведён исследовательский анализ данных (EDA) игровых гильдий. 
Цель исследования — изучить уровень развития и активности гильдий, проанализировать 
распределение ключевых игровых метрик и оценить, насколько данные подтверждают выдвинутые гипотезы.

## Используемые инструменты

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Выполненные задачи

* загрузка и объединение данных из нескольких источников;
* обработка пропущенных значений;
* анализ структуры данных;
* исследование распределений игровых метрик;
* визуализация данных с помощью гистограмм и boxplot;
* анализ показателей развития и активности гильдий.

## Исследованные метрики

* суммарная энергия гильдии;
* количество талантов;
* уровень гильдии;
* количество выполненных миссий;
* участие в войнах;
* участие в сезонах;
* средний рейтинг;
* среднее место;
* количество добавленных и удалённых участников.

## Результат

На основе исследовательского анализа были выявлены особенности распределения ключевых метрик, 
оценены различия между показателями развития и активности гильдий и сформулированы выводы по исследуемым гипотезам.
