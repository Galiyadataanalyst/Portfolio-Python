# Анализ активности игровых гильдий

## Описание проекта

В проекте проведён исследовательский анализ данных (EDA) игровых гильдий. Цель исследования — изучить показатели развития и активности гильдий, выявить особенности распределения ключевых игровых метрик, проверить выдвинутые гипотезы и определить признаки, характеризующие различные уровни активности гильдий.

## Используемые инструменты

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Выполненные задачи

* загрузка и объединение данных из нескольких источников;
* обработка пропущенных значений;
* исследовательский анализ данных (EDA);
* формирование агрегированных показателей по каждой гильдии;
* анализ распределения игровых метрик;
* визуализация данных с использованием гистограмм, boxplot и столбчатых диаграмм;
* сегментация гильдий по квартилям суммарной накопленной энергии;
* сравнение игровых показателей между выделенными группами активности;
* анализ платящих создателей гильдий и показателя ARPPU по уровням активности.

## Исследованные метрики

* суммарная энергия гильдии;
* количество талантов;
* уровень гильдии;
* количество выполненных миссий;
* участие в войнах;
* участие в сезонах;
* средний рейтинг;
* среднее место;
* количество добавленных и удалённых участников.

## Результаты

* Сформированы агрегированные показатели активности для каждой гильдии.
* Выполнена сегментация гильдий по квартилям суммарной накопленной энергии на группы с различным уровнем активности.
* Проведено сравнение игровых метрик между полученными группами.
* Установлено, что с увеличением уровня активности возрастает накопленная энергия, уровень гильдий и интенсивность участия в игровых событиях.
* Проанализированы различия в доле платящих создателей гильдий и значениях ARPPU между выделенными группами.
